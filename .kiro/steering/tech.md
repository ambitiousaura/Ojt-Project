# Technical Stack

## Core Technologies

- **HTML5**: Semantic markup with accessibility attributes
- **CSS3**: Modern layout with Grid and Flexbox
- **Vanilla JavaScript (ES6+)**: No frameworks or build tools required

## Browser APIs Used

- LocalStorage for data persistence
- ES6 Modules (`type="module"`)
- DOM APIs for rendering and event handling

## Key Libraries

None - this is a zero-dependency project.

## Development Approach

- No build system required
- No transpilation or bundling
- Direct browser execution
- ES6 module imports

## Running the Application

Simply open `indexes.html` in a modern browser. No build or compilation steps needed.

## Testing

The application includes a built-in demo mode accessible via the "Run demo tests" button in the settings panel. This generates 12 sample tasks for testing search, filters, and pagination.

## Browser Compatibility

Requires a modern browser with ES6 module support (Chrome 61+, Firefox 60+, Safari 11+, Edge 16+).
