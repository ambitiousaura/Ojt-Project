# Product Overview

SYNCLY is a framework-free task management application focused on performance, accessibility, and maintainability.

## Core Features

- Task CRUD operations (create, read, update, delete)
- Task completion tracking with visual indicators
- Search functionality with debounced input
- Multiple sorting options (newest, A-Z, Z-A, completed first)
- Filter views (all, active, completed)
- Pagination (5 tasks per page)
- Undo/redo functionality with state snapshots
- JSON export/import for data portability
- Multi-language support (English, Hindi)
- Demo mode for testing

## Design Philosophy

- No frameworks or libraries - pure vanilla JavaScript
- Accessibility-first with ARIA labels and semantic HTML
- LocalStorage for data persistence
- Modular ES6 architecture
- Clean, modern UI with responsive design
